## Descrição do Projeto

Este projeto é um portfólio profissional desenvolvido por Joel Júnior, com o objetivo de apresentar suas habilidades, experiências e projetos realizados na área de desenvolvimento web. O portfólio é uma vitrine digital que permite aos visitantes conhecerem melhor o trabalho do desenvolvedor.

## Tecnologias Utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS
- GSAP
- Zod
- React Hook Form

## Funcionalidades do Projeto

- `Apresentação de projetos`: O portfólio exibe uma seção de projetos desenvolvidos por mim, incluindo descrições, imagens, tecnologias utilizadas e links para os repositórios no GitHub.
- `Experiências Profissionais`: Uma seção que detalha minhas experiências anteriores, cargos ocupados e responsabilidades desempenhadas.
- `Tecnologias Dominadas`: Uma lista das principais tecnologias e ferramentas que utilizo no meu dia a dia como desenvolvedor.
- `Seção Sobre Mim`: Uma seção dedicada a apresentar minha trajetória profissional, habilidades técnicas e interesses na área de desenvolvimento web.
